# Tic Tac Toe Game (C++)

## Project Description

This is a console-based Tic Tac Toe game developed using C++. It demonstrates the use of arrays, loops, conditional statements, functions, and basic game logic.

## Features

- Two-player gameplay
- Dynamic board display
- Win detection
- Draw detection
- Input validation
- Replay option
- Menu-driven gameplay

## Technologies Used

- C++
- Object-Oriented Concepts
- Arrays
- Functions
- Loops
- Conditional Statements

## How to Run

Compile:

```bash
g++ tic_tac_toe.cpp -o game
```

Run:

```bash
./game
```

Windows:

```bash
game.exe
```
